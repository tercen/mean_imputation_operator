# Mean imputation operator

##### Description

`mean_imputation` operator replaces missing values by the column mean.

##### Usage

Input projection|.
---|---
`y-axis`        | numeric, input data, per cell 

Output relations|.
---|---
`median`        | numeric, median of the input data
